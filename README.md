# orchive_asr

Whisper ASR text and json transcripts for entire Orchive are saved on Animal Speech Synology here:
/cluster_DeepAL/data/AlexBarnhill/OrchiveWhisper

Metadata summary of all transcripts are saved here:
/data_summary/orchive_metadata_summary.csv

The "data_summary" directory includes other files that were used in the analysis and extraction steps, which may be useful as an overview, or for certain processing tasks (e.g., all text transcripts combined to a single file; speech duration summaries; locations used and similar ASR terms which were mapped to known locations; "proper nouns" and their frequency within transcripts; bar plot of recordings per year in Orchive; histogram of transcript length by word count, table of transcripts with word counts)

Overall summary of Orchive:
- Total files 23,508
- Total size 10.93 TB
- Total duration 18,806.4 h
- Equivalent days 783.6
- Speech time detected 2,209 h
- Speech proportion 11.75%
